Avalore Character Tracker

Welcome to the Avalore Character Tracker! This is a community run resource to better help with planning out and tracking builds. You can check out the main GitHub page below for more information, or to submit changes to the ever-updating feats, traits, and spells of the Avalore Server.

Save to the cloud with the "Save" button on the top right hand side of the page, and share this page with others with the Share Button. Your character's UUID (listed next to your character name) can be used to import this sheet to a new device, so be sure to save that share link somewhere if you want to use the sheet long term!
